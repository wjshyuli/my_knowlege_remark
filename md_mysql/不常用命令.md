# DDL 数据库操作

## 1. DDL数据库操作

-   查询

  `show databases;`            查询所有数据库

  `select database（）;`   查询当前数据库

  

-  创建    `create database;`

-  删除    `drop database;`

-  使用    `use 数据库名;`

## 2. DDL表操作

- 查询
  - `show tables;`  
  - `desc 表名;`      查询表结构
  - `show create table 表名;`      查询指定表的建表语句



- 创建

  ```
  create table 表名（
  	字段1  类型  [comment 注释]，
  	字段2  类型  [comment 注释]      # 最后一个,不能加
  ）；
  ```
  
  

- 数据类型

  > 数值类型：int ；bigint；float；double(4,1)；decimal(10,2)      无符号取值，不取负数，UNSIGNED
  >
  > 字符串:	char(10); varchar(10) ;  blob;text
  >
  > 日期时间：
  >
  > - DATE :YYYY-MM-DD
  > - TIME: HH:MM:SS
  > - YEAR:YYYY
  > - DATETIME
  > - TIMESTAMP

- 修改

  添加     `ALTER 表名 ADD 字段名 类型(长度)[COMMENT 注释][约束]`  （`ALTER emp add name varchar(20)`）
  
   `

- 修改数据类型

  - `ALTER TABLE 表名 MODIFY 字段名 新数据类型（长度）；`

- 修改字段名和字段类型

  - `ALTER TABLE CHANGE 旧字段名 新字段名 类型（长度）[COMMENT 注释][约束]；`

- 修改表名

  - `ALTER TABLE 表名 RENAME TO 新表名 ;`

- 删除字段

  - `ALTER TABLE 表名 DROP 字段名;`

- 删除表

  - `DROP TABLE [IF EXISTS] 表名 ；`
