# DDL 数据库操作

## 1. DDL数据库操作

-   查询

  `show databases;`            查询所有数据库

  `select database（）;`   查询当前数据库

  

-  创建    `create database;`

-  删除    `drop database;`

-  使用    `use 数据库名;`

## 2. DDL表操作

- 查询
  - `show tables;`  
  - `desc 表名;`      查询表结构
  - `show create table 表名;`      查询指定表的建表语句



- 创建

  ```
  create table 表名（
  	字段1  类型。[comment 注释]，
  	字段2  类型。[comment 注释]      # 最后一个,不能加
  ）；
  ```
